# Лабораторна робота №4 — Робота з JavaScript

Резюме Jacob Hancock з виконанням усіх пунктів ЛР4.

## Що зроблено

1. **localStorage** — зберігання інформації про ОС та браузер, відображення у футері
2. **API** — завантаження коментарів з `jsonplaceholder.typicode.com/posts/{id}/comments`
3. **Модальне вікно** — показ форми зворотнього зв'язку через 1 хвилину, відправка через Formspree
4. **Тема** — переключатель день/ніч та автоматичне перемикання (07:00–21:00 — день, решта часу — ніч)

## Як запустити

Відкрийте `index.html` у браузері (або через локальний сервер).

## Що потрібно налаштувати

1. **Formspree** — зареєструйтесь на [formspree.io](https://formspree.io), створіть форму і замініть `YOUR_ENDPOINT` у `index.html` на отриманий endpoint.
2. **Номер варіанту** — у `script.js` у рядку `const VARIANT_NUMBER = 5` замініть `5` на свій номер у журналі групи.
